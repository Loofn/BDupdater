## Installation
**with PIP**
```
pip install bdupdater
```

**manually**
```
git clone https://github.com/Loofn/BDupdater.git
cd /path/to/bdupdater
python setup.py sdist bdist_wheel
pip install .
```

## Usage
```
bdupdater update - Updates Discord and BetterDiscord.
```
